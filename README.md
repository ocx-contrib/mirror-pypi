# mirror-pypi

OCX mirror for PyPI applications (pycowsay, black, …), published to `dev.ocx.sh`
as pylock-resolved OCX packages. WIP — the whole pipeline (dev-channel binary,
corpus specs, CI) is being built out against the `dev.ocx.sh` dev channel.
